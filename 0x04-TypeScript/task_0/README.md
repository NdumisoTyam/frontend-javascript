# 📘 Task 0 — Student Table with TypeScript

## 🚀 Project Overview

This project uses TypeScript and vanilla JavaScript to dynamically render a table showing student details. It leverages an interface to define student structure and manipulates the DOM to display each student's first name and location.

---

## 🛠️ Tech Stack

- TypeScript
- Webpack
- ESLint
- Vanilla JavaScript (DOM manipulation)
- Node.js
